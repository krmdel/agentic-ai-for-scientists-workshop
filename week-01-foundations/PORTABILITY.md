# Portability — Agentic AI Workshop demo

How to take the workshop demo (dashboard project + Obsidian vault) from this
laptop to the presentation laptop without re-generating everything from scratch.

---

## What's portable

| Artifact | Path | Travels via |
|---|---|---|
| Dashboard project (papers + hypothesis + data + figures + draft + runs) | `projects/briefs/agentic-ai-workshop/` | rsync or `git add -f` |
| Obsidian vault | `projects/briefs/agentic-ai-workshop/obsidian-vault/` | rsync; opens via *File → Open folder as vault* |
| Workshop slides + brief | `projects/briefs/agentic-ai-workshop/` | already in git |

Both `projects/*` and `projects/briefs/*` follow Organon's gitignore rule (project
outputs are local-only). So you have **two options** for moving them.

---

## Option A — rsync (recommended for first-time setup)

From this laptop, with the presentation laptop reachable on the LAN:

```bash
# 1. Dashboard project
rsync -av --delete \
  ~/Projects/organon/projects/briefs/agentic-ai-workshop/ \
  presentation-laptop:~/Projects/organon/projects/briefs/agentic-ai-workshop/

# 2. Obsidian vault (lives under the brief, separately portable)
rsync -av --delete \
  ~/Projects/organon/projects/briefs/agentic-ai-workshop/obsidian-vault/ \
  presentation-laptop:~/Projects/organon/projects/briefs/agentic-ai-workshop/obsidian-vault/

# 3. Brief + slides (already in git, but rsync covers them in one pass)
rsync -av \
  ~/Projects/organon/projects/briefs/agentic-ai-workshop/ \
  presentation-laptop:~/Projects/organon/projects/briefs/agentic-ai-workshop/
```

Replace `presentation-laptop:` with the actual hostname or IP. If both machines
are signed into iCloud / Dropbox, you can also drag the two folders into the
synced area and skip the rsync. The vault is intentionally separate from the
user's main iCloud Obsidian vault — copy-don't-move.

---

## Option B — force-commit to a branch and pull on laptop #2

If the presentation laptop is offline:

```bash
# This laptop:
git checkout -b workshop-demo-snapshot
git add -f projects/briefs/agentic-ai-workshop/ \
         projects/briefs/agentic-ai-workshop/obsidian-vault/
git commit -m "snapshot: workshop demo project + vault"
git push origin workshop-demo-snapshot

# Presentation laptop:
git fetch origin
git checkout workshop-demo-snapshot
```

> **Caveat**: the `projects/*` and `obsidian-vault/` paths are normally gitignored.
> The `-f` flag forces inclusion for this snapshot branch only. After the workshop,
> drop the branch — don't merge into `main`.

## On the presentation laptop, first run

1. **Start the dashboard** from the dashboard folder:
   ```bash
   cd ~/Projects/organon/projects/briefs/organon-dashboard
   npm run dev
   # → opens at http://localhost:8769
   ```
2. **Switch project picker to** `Agentic Ai Workshop` (the dashboard auto-discovers
   any non-hidden directory under `projects/`; `listProjects()` does the scan).
3. **Open Obsidian** → *File → Open folder as vault* → select
   `~/Projects/organon/projects/briefs/agentic-ai-workshop/obsidian-vault/`.
4. Press `Cmd+G` for the graph view. Colour groups are pre-configured in
   `.obsidian/graph.json` (identity orange · concepts gold · paper-notes blue ·
   experiments red · data-notes teal · drafts yellow · daily purple · inbox grey).

  Runs workspace shows traffic.

## After the workshop — cleanup options

- *Keep as a public demo*: move under `projects/briefs/` and update CLAUDE.md.
  The cohort CSV is synthetic and shareable.
- *Strip it out*: `rm -rf projects/briefs/agentic-ai-workshop/` and
  `rm -rf projects/briefs/agentic-ai-workshop/obsidian-vault/`. The brief, slides,
  and `_build_*.py` scripts remain in git so you can regenerate later.

## Regenerating from scratch

If you need a fresh build (different cohort, different figure), the two scripts
are deterministic except for the figure (Gemini outputs vary):

```bash
python3 projects/briefs/agentic-ai-workshop/_build_demo.py        # papers + hypothesis + cohort + manuscript + runs
python3 projects/briefs/agentic-ai-workshop/_build_vault.py  # obsidian vault

# Figure (regenerate):
uv run .claude/skills/viz-nano-banana/scripts/generate_image.py \
  --prompt "$(jq -r .params projects/briefs/agentic-ai-workshop/figures/fig-*/index.json | head)" \
  --filename "v1.png" --resolution "2K" --aspect-ratio "4:3"

python3 projects/briefs/agentic-ai-workshop/_finalize.py          # wire figure in
```
